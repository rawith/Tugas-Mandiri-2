# Tugas Mandiri 2 - Pengantar Data Sains - Analisis Data Calon TNI

## Deskripsi Project
Project ini menganalisis data tinggi badan calon TNI menggunakan Python. Analisis meliputi:
- Statistik deskriptif
- Visualisasi data
- Kategorisasi tinggi badan
- Deteksi outlier

## Dataset
File: `calon_tni.csv`
Berisi data 10 calon TNI dengan kolom:
- name: Nama calon
- alamat: Alamat calon  
- tinggi: Tinggi badan dalam cm

## Requirements
- Python 3.6+
- pandas
- numpy
- matplotlib
- seaborn

## Cara Menjalankan
1. Install requirements: `pip install pandas numpy matplotlib seaborn`
2. Run script: `python analisis_calon_tni.py`
3. Atau buka Jupyter Notebook: `analisis_calon_tni.ipynb`

## Hasil Analisis
- Rata-rata tinggi: 175.70 cm
- Tertinggi: 185 cm
- Terendah: 160 cm
- Distribusi kategori: Tinggi (4), Sedang (4), Pendek (2)
